# react-electron-app
Boilerplate React Electron App

Requirements:
Node.js NPM Yarn

Installing:
$git clone this
cd react-electron-app
npm install

How to use:
//TO DEVELOPMENT
npm run electron-dev

//TO BUILD/DIST
npm run electron-pack
